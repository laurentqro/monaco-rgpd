# Monaco RGPD

Plateforme SaaS de conformité RGPD pour Monaco - Conforme à la Loi n° 1.565

## Stack Technique

- Rails 8.0.3
- PostgreSQL 18
- Inertia.js + Svelte 5
- Tailwind CSS

## Démarrage

```bash
bin/setup
bin/dev
```
